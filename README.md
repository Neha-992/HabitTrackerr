# 🌟 HabitTrackerr

A full-stack Habit Tracker Web Application built using Django that helps users build positive habits, track consistency, monitor progress, and stay productive through gamification and analytics.

---

# 🚀 Features

- 🔐 User Registration & Login Authentication
- 📝 Add, Edit & Delete Habits
- 🔥 Daily Streak Tracking System
- 🎮 XP & Level Gamification System
- 📊 Weekly Progress Analytics Chart
- 📈 Activity Heatmap
- 📂 CSV Export Functionality
- 🔍 Search & Filter Habits
- 🌙 Dark Mode Support
- 📱 Fully Responsive Design
- 🏆 Leaderboard System

# ⚡ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Neha-992/HabitTrackerr.git

2️⃣ Navigate to Project Folder
cd HabitTrackerr

3️⃣ Create Virtual Environment
python -m venv venv

4️⃣ Activate Virtual Environment (Windows)
venv\Scripts\activate

5️⃣ Install Django
pip install django

6️⃣ Run Migrations
python manage.py makemigrations
python manage.py migrate

7️⃣ Start Development Server
python manage.py runserver
